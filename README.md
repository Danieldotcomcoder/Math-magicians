## Math magicians
 It is a Single Page App that allows users to use a normal calculator.

## Built With
### Major languages
- React
- CSS
- Javascript
### Technologies used
- Github
- Gitflow
- Visual Studio Code

## Getting Started
To get a running local copy follow these simple example steps:
- git clone  https://github.com/Danieldotcomcoder/Math-magicians.git

Run: 
-  npm install
-  npm run build.
-  npm start.
